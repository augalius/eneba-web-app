# Eneba Web App

Simple full-stack web application with fuzzy search capability that stores 3 games: Fifa 23, Red Dead Redemption 2 and Split Fiction.

# Tech-Stack
- React
- Go
- postgreSQL
- Docker

## Running app locally
1. Clone the repository
2. Open root `cd eneba-web-app`
3. Build and run with Docker Compose `docker-compose up --build`
4. Open app on `http://localhost:3000`
